Getting Started

Welcome to Ancroo, the self-hosted AI workflow platform.

Key features:

  • Browser extension for text transformation
  • Speech-to-text via Whisper
  • Automation workflows with n8n

Visit the documentation to learn more.

How to use: Select the formatted text above, open the Ancroo side panel, select "HTML to Markdown", and click Execute. The HTML is sent to Ancroo Runner for conversion and the Markdown is copied to your clipboard.
